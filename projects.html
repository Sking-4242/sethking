Archon Professional

Open-source visual cloud architecture & IaC platform
Active
AWS Azure GCP On-Prem Terraform Multi-LLM AI

Archon Professional is a visual cloud infrastructure design tool: drag, drop, and wire together AWS, Azure, GCP, and on-prem components — over 260 in the library — directly on a canvas, then generate production-ready Terraform from the diagram using any of five integrated LLM providers.

Highlights

  • Terraform import. Reverse-engineers existing .tf files (single or multi-file projects) back into an editable canvas — preserving VPC/subnet/resource nesting, security groups, IAM roles, and network connections — then generates an import report summarizing every resource action taken.
  • Built-in compliance scanning. Validates architectures against CIS, SOC 2, PCI DSS, HIPAA, and NIST CSF, surfacing critical/warning/info findings (e.g. publicly exposed EKS endpoints, unencrypted secrets) with remediation guidance for each.
  • IAM & security group management. Dedicated tabs to review roles and policy attachments, and to flag components with no IAM role assigned.
  • Live cost estimation. Real-time pricing from AWS/Azure/GCP APIs (with static fallback and a 1-hour TTL cache), editable usage assumptions per resource, growth-adjusted 6/12/24-month cost projections, and a FinOps mode that compares modeled cost against actual AWS Cost Explorer spend.

Archon Academy

Cloud certification exam-prep platform, companion to Archon Professional
Active
AWS Azure GCP E-Learning Exam Prep

Archon Academy is a structured study platform for AWS, Azure, and GCP certifications — built to teach the same cloud concepts that Archon Professional lets you design with. It pairs long-form lessons with domain-weighted cert paths, hands-on practice, and full-length timed exams, so studying stays tied to a concrete exam blueprint instead of scattered notes.

Highlights

  • Cert paths mapped to real exam blueprints. Each path (e.g. AWS Cloud Practitioner CLF-C02) tracks exam readiness against scored-question counts, time limits, and passing thresholds, broken into weighted domains like Cloud Concepts (24%) and Security and Compliance (30%).
  • 26+ modules across AWS, Azure, and GCP. Filterable by difficulty (Beginner through Expert) and by certification tag (CCP, SAA, SCS, SOA, DVA, DEA, DBS, DOP, MLA, MLS, AZ-900, GCP CDL, and more), each with lesson counts and progress tracking.
  • In-depth written lessons. Every lesson explains not just the "what" but the "why" — grounding concepts in real-world problems (e.g. the capacity mismatch problem behind cloud elasticity) and tying them back to specific exam domains.
  • Practice exams and quizzes. Full 90-question timed practice tests per certification with multiple difficulty variants (below/at/above exam level), plus a study mode with unlimited time and per-question explanations and instant correct/incorrect feedback.
